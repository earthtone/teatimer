#grunt2015
=========
##Grunt/Babel Boilerplate for Front-End Web Projects

### dependencies:
- Node
- Grunt
- Bower
- nodeSass

### devDependencies
- grunt babel
- grunt contrib jshint
- grunt contrib connect
- grunt contrib watch
- grunt contrib concat
- grunt sass
- grunt contrib handlebars
- grunt contrib copy
- jQuery
- Handlebars

### commands
local/dev: 'grunt' => 'grunt build', 'grunt serve'
remote/deploy: 'grunt build'
